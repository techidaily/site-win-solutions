Unlock These 10 Tips for Getting ‘Stray’ Running This Year

Stray is finally here! While users enjoy this adventure game, some reported that they fail to launch Stray or get crashing at startup. But don’t be upset. This post will introduce 10 fixes you can try to tackle the Stray not launching issue.

How to fix Stray not launching?

  1. Run the game as an administrator
  2. Update graphics driver
  3. Verify integrity of game files
  4. Repair system files
  5. Disable overlay
  6. Update DirectX
  7. Install Microsoft Visual C++ Redistributables
  8. Close background processes
  9. Update Windows
  10. Turn off firewall and antivirus temporarily

You don’t have to try them all. Just work your way down the list until you find the one that works for you.

Before we start

Check the system requirements of Stray before digging into fixes. It happens that the game is freezing, lagging, or looks like a slideshow when your computer hardware fails to meet its minimum system requirements.

Here are the minimum and recommended system requirements for Stray:

Processor Intel Core i5-2300 | AMD FX-6350
Memory 8 GB RAM

If your PC doesn’t suit the needs of the minimum requirements, try to update your hardware first. If you’re confident that your hardware rig fits the game perfectly, start troubleshooting the Stray not launching with the listing fixes.

Fix 1Run the game as an administrator

Whatever games you’re playing, it’d be best to run the game as an administrator. That’s because an administrator is usually considered authorized. Therefore, all the operations will get full support and maximum resources of the system. Let’s see how to run Stray as an administrator once-for-all:

  1. Right-click theStray.exe file on your computer and clickProperties from the pop-up list.
  2. ClickCompatibility . Then check the boxRun this program as an administrator , and clickApply andOK to save changes.
  3. Now you can launch the game in the way you normally do.

Check if the game still has the launching or crashing problem. If this fix doesn’t work for you, try the next one.

Fix 3 Verify integrity of game files

One of the possible causes of game launching issues is missing or corrupted game files. Luckily, many PC clients allow you to verify the integrity of game files through the library:

  1. Open Steam and clickLibrary .
  2. Right-clickStray and selectProperties .
  3. SelectLOCAL FILES on the left and clickVerify integrity of game files…

Steam will run its program to verify the files of this game for you. Once done, exit the software client and launch it again. Have a test to see if the launching issue is gone.

Fix 5 Disable overlay

If you’re using overlay apps such as Discord or Xbox, try to disable them. Because it’s reported that these apps may conflict with Stray, leading to crashing or freezing. What’s more, some games have a problem pairing with Steam overlay. So disabling Steam overlay may also be a solution for you:

  1. Open Steam client.
  2. Navigate to theSteam >Settings >In-game tab.
  3. UncheckEnable the Steam Overlay while in-game .
  4. Relaunch Steam.

Once you have disabled apps and Steam, open the game for checking the Stray startup issues.

Fix 6 Update DirectX

DirectX 12 is listed as one of the system requirements for Stray. If you’re using a lower version, things may go wrong, and launching problems in Stray can happen. Follow the steps to check your DirectX version:

  1. Press theWindows logo key andR on the keyboard to invoke the Run box.
  2. Typedxdiag and clickOK .
  3. Check your DirectX version in the pop-up window.

If DirectX 12 is equipped on your PC, you can continue to tryFix 7 .

However, check the following guide toupdate DirectX if you’re using a lower version.

  1. Typecheck in the Windows search box. Then click to openCheck for updates .
  2. ClickCheck for updates .

Follow the on-screen instructions so that Windows update can download and install DirectX automatically for you.

Fix 7 Install Microsoft Visual C++ Redistributables

The latest version of Microsoft Visual C++ Redistributables ensures that Stray and other PC games can run properly. Check your Microsoft Visual C++ version first:

  1. Typecontrol in the Windows search box. OpenControl Panel .
  2. ClickPrograms .
  3. ClickPrograms and Features .
  4. Check your Microsoft Visual C++ Redistributable version.

If you find your Microsoft Visual C++ Redistributables are outdated, head toMicrosoft website to get it updated.

Click specific link to download the latest Microsoft Visual C++ Redistributable packages. After that, open the downloaded file and clickRepair .

Restart your computer to make changes apply.

Fix 8 Close background processes

Stray not launching easily occurs when there are too many programs running in the background. That’s because few system resources are distributed to the game. To solve this, you can close unwanted processes and set the game with high priority:

  1. Right-click the Windows taskbar and clickTask Manager .
    Project Manager - Asset Browser for 3Ds Max
  2. Choose the processes consuming too many resources, and then clickEnd task .
  3. Click theDetail tab. Right-clickStray.exe and set its priority toHigh .

Return to the game to check if the problem is solved.

Fix 9 Update Windows

Downloading and installing all available Windows updates is a common solution when encountering program issues. The reason is that the latest Windows components tend to fix bugs crippling the proper working of applications. Here is how to update Windows:

  1. On your keyboard, hit theWindows logo key andI to invoke Settings. Then clickUpdate & Security .
  2. ClickCheck for updates .

Still no luck? Try the final fix below.
