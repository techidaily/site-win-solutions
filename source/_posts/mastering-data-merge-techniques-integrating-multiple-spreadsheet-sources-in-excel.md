##  Consolidate Spreadsheet Data in Excel

 Microsoft Excel offers a Consolidate feature that can tackle this task. It allows you to combine data from different sheets into one spreadsheet. Plus, you can [choose the function](https://win11.techidaily.com/renaissance-pc-refresh-with-atlasos/) you want to use from options like [SUM](https://instagram-videos.techidaily.com/updated-steps-to-instagram-verification-and-fan-growth-in-under-150-characters/), [AVERAGE](https://screen-video-capture.techidaily.com/updated-in-2024-mastering-your-first-zoom-webinar-with-ease/), PRODUCT, and [COUNT](https://howto.techidaily.com/fix-the-error-of-unfortunately-the-processcomandroidphone-has-stopped-on-vivo-v29-pro-drfone-by-drfone-fix-android-problems-fix-android-problems/).

 To demonstrate the process, we'll combine [expense sheets](https://fox-blue.techidaily.com/updated-diving-into-the-depths-with-gopro-hero5-for-2024/) from three employees into one final sheet that totals all expenses. As you'll see, each sheet is set up the same way which makes the consolidation much simpler.

 Go to the sheet where you want to pull in the data from the other sheets. Select a cell to insert the combined data. If you have a cell range, you can choose the upper-left cell.

![Select a cell for the combined data](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/SelectCell-ExcelConsolidate.png) 

 Head to the Data tab and the Data Tools section of the ribbon. Click "Consolidate."

![On the Data tab, click Consolidate](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/DataConsolidate-ExcelConsolidate.png) 

 When the Consolidate window opens, start by selecting the Function you want to use from the drop-down list. For our example of adding expenses, we choose "Sum."

![Choose a Function](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/Function-ExcelConsolidate.png) 

 Next, use the Reference box to obtain the first sheet and cell range. You can simply move to that sheet and [select the cells](https://buynow-tips.techidaily.com/exploring-a-ravaged-world-on-motorcycle-in-days-gone-our-comprehhavis-review/). The Consolidate window remains open as you do this. For our example, we go to Joe's sheet and select cells A1 through B5.

![Select the sheet and cells](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/Reference-ExcelConsolidate.png) 

 When the cell reference appears, click "Add." This places it into the All References list.

![Select the sheet and cells](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/Reference-ExcelConsolidate.png) 

 Click inside the Reference box and then go to the next sheet you want to add and select the cells from it. Then, click "Add" to place it into the list.

![Add more References](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/AddAllReferences-ExcelConsolidate.png) 

 Continue this same process for all sheets and cell data. For our example, we gathered data from the sheets labeled Joe, Jane, and Jim into our Final spreadsheet.

![All References added](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/AddedAllReferences-ExcelConsolidate.png) 

 Before you finish, you can check the boxes at the bottom for using the labels and creating links to the source data if you like. Click "OK." To make our job easier, we'll [use labels](https://android-pokemon-go.techidaily.com/hacks-to-do-pokemon-go-trainer-battles-for-huawei-p60-drfone-by-drfone-virtual-android/) for the top row and left column.

![Add labels or links](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/LabelsLinks-ExcelConsolidate.png) 

 You should then see the data consolidated into your sheet.

![Combined data in Excel](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2021/11/DataCombined-ExcelConsolidate.png)
