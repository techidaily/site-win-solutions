##  Add Times in Microsoft Excel

 You might be keeping track of the time you spend working on a task or project. When that work is complete, you'll want the total time you spent. Whether for your own records or for billing a client, you can use a combination of formulas and formatting to get accurate time totals.

###  Format the Cells for Hours and Minutes

###  Format Times Using 24 Hours

 When you're adding entries that use 24-hour time, it's quite possible that you'll get an incorrect total. To avoid this, you can format the cell containing the result.

 In the Format Cells window, make sure that the Number tab is selected. Pick "Custom" as the Category and select "\[h\]:mm;@" from the list, or enter the format into the Type box.
